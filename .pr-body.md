## P0-01: Contracts package

### Goal
Define immutable API and schema contracts used by runtime, telemetry, and gate checker services.

### Changes
- **contracts/openapi/inference.yaml** — OpenAI-compatible steering inference API with `SteeringParams` (concept, preset, layers, multiplier, profile_id), profile retrieval, and health check endpoints. Includes bearer auth security scheme.
- **contracts/schema/profile.json** — Versioned steering profile schema (base model, layers, preset table, vector bundle, judge bundle).
- **contracts/schema/run-metadata.json** — Per-run metadata schema with all required fields from feedback-loop.md section 4 (env, agent, model, profile, layers, preset, multiplier, concept, request/thread IDs, git SHA, judge bundle, latency, tokens, degenerate, language_shift).
- **contracts/schema/experiment-decision.json** — Champion/challenger experiment outcome schema with hard gates, scores, rank score, and promotion decision.
- **contracts/fixtures/** — Valid and invalid fixtures for all three schemas.
- **scripts/contracts-lint.mjs** — JSON Schema 2020-12 meta-validation.
- **scripts/contracts-test.mjs** — Fixture validation against schemas.
- **package.json** — Added `contracts:lint`, `contracts:test` scripts and devDependencies (`@redocly/cli`, `ajv`, `ajv-formats`). Updated `verify` to include contract checks.
